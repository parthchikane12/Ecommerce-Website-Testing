# 🧪 E-commerce Website Testing Project

## 📌 Project Overview

This project demonstrates **manual testing of an E-commerce web application**.
It includes test planning, test cases, defect reports, and a test summary report.

The objective of this project is to practice **Software Testing Life Cycle (STLC)** and create real-world testing documentation.

---

## 🎯 Modules Tested

* User Registration
* Login
* Product Search
* Add to Cart
* Checkout
* Payment

---

## 🧰 Testing Type

* Functional Testing
* UI Testing
* Negative Testing
* Regression Testing

---

## 📁 Project Files

| File Name            | Description                              |
| -------------------- | ---------------------------------------- |
| TestPlan.md          | Describes the testing strategy and scope |
| TestCases.md         | Contains detailed test cases             |
| BugReport.md         | Lists defects found during testing       |
| TestSummaryReport.md | Final testing report                     |
