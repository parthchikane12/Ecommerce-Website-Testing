# Test Plan – E-commerce Website

## Objective

To verify that the E-commerce website works correctly and meets functional requirements.

## Scope

Testing the following modules:

1. User Registration
2. Login
3. Product Search
4. Add to Cart
5. Checkout
6. Payment

## Testing Type

* Functional Testing
* UI Testing
* Regression Testing
* Negative Testing

## Entry Criteria

* Website is accessible
* Test environment ready

## Exit Criteria

* All critical test cases executed
* Major defects fixed
